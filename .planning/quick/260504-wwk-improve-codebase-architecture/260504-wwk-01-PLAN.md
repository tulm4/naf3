<objective>
Extract NssaaStatus state machine into domain layer and wire real Prometheus metrics into biz/router.go.
</objective>

<context>
From arch-analysis and research findings:

**Problem 1:** `internal/types/nssaa_status.go` has helper methods (`IsTerminal`, `IsPending`) but no state transition logic. State transitions are implicit in EAP engine.

**Problem 2:** `internal/biz/router.go` has a no-op `Metrics` struct — `RecordAAARequest` and `RecordAAALatency` are empty functions.

**Solution:** Create `internal/domain/nssaa_status.go` with explicit state machine. Wire `metrics.AaaRequestsTotal` into biz Metrics.

@internal/types/nssaa_status.go
@internal/biz/router.go
@internal/metrics/metrics.go
</context>

  <action>
Create `internal/domain/nssaa_status.go` with:

1. Package doc and spec reference (TS 29.571 §5.4.4.60)
2. Type alias: `type NssaaStatus = types.NssaaStatus`
3. Re-export constants: `StatusNotExecuted`, `StatusPending`, `StatusSuccess`, `StatusFailure`
4. `AuthEvent` type (iota-based): `EventAuthStarted`, `EventEAPRound`, `EventAAAComplete`, `EventAAAFailed`
5. `TransitionError` struct with `Error()` method
6. `TransitionTo(current NssaaStatus, event AuthEvent) (NssaaStatus, error)` function implementing:
   - NOT_EXECUTED + EventAuthStarted → PENDING
   - PENDING + EventAAAComplete → EAP_SUCCESS
   - PENDING + EventAAAFailed → EAP_FAILURE
   - PENDING + EventEAPRound → PENDING (intermediate round, no error)
   - Terminal states (SUCCESS/FAILURE) absorb all events (return current, nil)
   - Invalid transitions return `&TransitionError{From: current, To: "invalid_event"}`
</action>

  <done>NssaaStatus state machine extracted with validated transitions</done>
</task>

In `internal/biz/router.go`:

1. Add import for `"github.com/operator/nssAAF/internal/metrics"`
2. Replace empty `Metrics` struct with:

```go
type Metrics struct {
    requestsTotal *prometheus.CounterVec
    latencyHist   *prometheus.HistogramVec
}

func NewMetrics() *Metrics {
    return &Metrics{
        requestsTotal: metrics.AaaRequestsTotal,
        latencyHist:   metrics.AaaRequestDuration,
    }
}
```

3. Replace empty `RecordAAARequest` implementation:

```go
func (m *Metrics) RecordAAARequest(protocol, host, result string) {
    m.requestsTotal.WithLabelValues(protocol, host, result).Inc()
}
```

4. Replace empty `RecordAAALatency` implementation:

```go
func (m *Metrics) RecordAAALatency(protocol, host string, d time.Duration) {
    m.latencyHist.WithLabelValues(protocol, host).Observe(d.Seconds())
}
```

5. Update `NewRouter` to accept `*Metrics` via options and initialize default metrics:

```go
func WithMetrics(m *Metrics) RouterOption {
    return func(r *Router) { r.metrics = m }
}
```

6. In `NewRouter`, if `r.metrics == nil`, set to `NewMetrics()` as default.
</action>
